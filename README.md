
# PSUsphere

**PSUsphere** is a Django-based management system designed for the **Palawan State University** ecosystem. It provides a centralized platform to manage Colleges, Programs, Student Organizations, and Memberships.

## Description
This project streamlines the administration of student-led organizations. It features a robust backend for tracking student enrollments across different colleges and their respective organizational involvements, ensuring data integrity and easy accessibility for administrators.

### Key Features
- **Centralized Database**: Manage 8 Colleges and various academic Programs.
- **Organization Tracking**: Manage student groups like **ACS** and **SITE**.
- **Refactored Admin Interface**: 
    - Searchable student records by name and ID.
    - Filterable college and organization lists.
    - Custom "Program" display for organization members.
